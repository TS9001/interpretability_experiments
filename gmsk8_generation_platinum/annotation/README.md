# GMSK8 Test Set Annotation

This folder contains the first 100 examples from the GMSK8 test split.

## Files

- `first_100_test_examples.jsonl`: Examples in JSON Lines format (one per line)
- `first_100_test_examples.json`: Examples in standard JSON array format

## Dataset Information

- Total examples in this file: 100
- Source: GMSK8 dataset (test split)
- Fields: ['question', 'answer', 'cleaning_status']
